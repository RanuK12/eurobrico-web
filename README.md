# Eurobrico Web

Análisis y prototipo web de Eurobrico.com/it — plataforma de e-commerce de bricolaje y hogar en Italia.

## Contenido
- `analisis_eurobrico.md` — Análisis detallado del sitio actual
- `docs/` — Documentación adicional
- `prototype/` — Prototipo del proyecto
- `Index.html` — Página principal del prototipo

## Análisis cubierto
- Arquitectura de información y navegación
- Componentes funcionales (buscador, checkout, área cliente)
- Rendimiento y usabilidad
- Accesibilidad (falta de `alt`, contraste bajo)
- Recomendaciones de mejora

## Stack
- HTML/JS para el prototipo
- Node.js (package.json presente)
